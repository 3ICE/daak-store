How this Game store works

  • This app was deployed to Heroku, using Git. Based on the heroku python-getting-started project.
  • This game store allows you to play multiple games. Log in here, then you can access our game store.
  • When Heroku received the source code, it fetched all the dependencies in the requirements.txt file, creating a deployable slug.
  • The platform then spins up a dyno, a lightweight container that provides an isolated environment in which the slug can be mounted and executed.
  • Check out our Restful API https://daak-store.herokuapp.com/highscores/GAME ID HERE, for highscores for all users.
  • Or https://daak-store.herokuapp.com/highscore/GAME ID HERE/USER ID HERE, for highscores for one user in one game.
